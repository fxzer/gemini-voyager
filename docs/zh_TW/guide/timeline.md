# 時間旅行

長對話是災難。上上下下，找不著北。
Voyager 把對話變成一條線。

## 看見節奏

看螢幕右側。
每個點都是一句話。那是你對話的脈搏。

## 導航，一步到位

- **瞬移**：點哪去哪，絕不拖泥帶水。
- **偷看**：滑鼠放上去，不用跳轉也能看內容。
- **插眼**：長按節點 **加星**。給大腦打個書籤。
- **層級 (實驗性)**：右鍵點擊節點，設置不同層級（1-3 級）或摺疊子節點。讓深度分支對話一目了然。
- **調寬窄**：拖動時間軸內側邊緣，自由調整寬度。
- **快捷鍵**：用鍵盤飛速穿梭。默認 `j`/`k` 上下跳轉，`gg`/`GG` 直達首尾，想改就改。

## 插眼：給靈感釘個釘

長對話裡藏著金子。用星標把重要時刻釘住。

**怎麼用**：長按（約 0.5 秒）時間軸上任意節點，看到它亮起來——這條訊息就被加星了。

- 再次長按取消星標。
- 加星的節點會保持高亮，一眼就能找到。
- 點擊時間軸頂部的星標圖標，打開 **星標歷史** 面板，集中查看所有標記過的訊息。

把它想像成在對話中插旗——靈感迸發的瞬間、完美的回答、或者下次要從這裡繼續的位置。

::: tip 小貼士
開啟[雲端同步](/guide/cloud-sync)後，星標會跨裝置同步。
:::

## 鍵盤，更快

不想用滑鼠？用鍵盤。

**就像在 Gemini 裡開了 Vim Mode。**

### 默認快捷鍵

- `k` - 跳到上一個節點
- `j` - 跳到下一個節點
- `gg` - 跳到第一輪
- `GG` - 跳到最後一輪

### 自定義

打開擴充功能設置，點擊快捷鍵框，按下你想用的鍵。
任意鍵，任意組合。`n`/`p`？`,`/`.`？隨你。

**流動模式**下連按會排隊播放動畫。
**跳躍模式**下立即響應，速度拉滿。
