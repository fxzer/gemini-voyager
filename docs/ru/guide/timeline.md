# Путешествие во времени

Длинные разговоры могут быть запутанными. Вы прокручиваете вверх, прокручиваете вниз, теряете место.
Voyager превращает ваш разговор в таймлайн.

## Увидьте форму вашего чата

Посмотрите на правую часть экрана.
Каждый узел представляет сообщение. Таймлайн визуализирует ритм вашего диалога.

## Навигация решена.

- **Телепортация**: Нажмите на узел, чтобы мгновенно перейти к этому сообщению.
- **Предпросмотр**: Наведите курсор, чтобы увидеть содержимое, не двигаясь.
- **Закладка**: Длительное нажатие на узел, чтобы **пометить его звездочкой**. Это как закладка для вашего мозга.
- **Уровни (Экспериментально)**: Щелкните правой кнопкой мыши по узлу, чтобы установить различные уровни (1-3) или свернуть дочерние элементы. Идеально подходит для того, чтобы сделать разветвленные разговоры понятными.
- **Ширина**: Перетащите внутренний край таймлайна, чтобы изменить его ширину.
- **Клавиатура**: Перемещайтесь со скоростью мысли. По умолчанию `j`/`k`, `gg`/`GG` для перехода к первому/последнему ходу.

## Звёздочка: закладка для озарений

В длинных разговорах прячется золото. Отметьте звёздочкой важные моменты.

**Как это работает**: Долгое нажатие (~0.5 сек) на любой узел таймлайна. Увидите, как он засветится — это сообщение теперь помечено звёздочкой.

- Повторное долгое нажатие снимает звёздочку.
- Помеченные узлы остаются подсвеченными, чтобы вы могли найти их с одного взгляда.
- Нажмите на значок звёздочки в заголовке таймлайна, чтобы открыть панель **Истории звёздочек** и увидеть все отмеченные сообщения в одном месте.

Это как воткнуть флаг в разговор — момент озарения, идеальный ответ или точка, откуда вы хотите продолжить в следующий раз.

::: tip Совет
Звёздочки синхронизируются между устройствами, если включена [Облачная синхронизация](/guide/cloud-sync).
:::

## Еще быстрее с клавишами

Не хотите использовать мышь? Используйте клавиатуру.

**Это как включить режим Vim в Gemini.**

### Горячие клавиши по умолчанию

- `k` - Перейти к предыдущему узлу
- `j` - Перейти к следующему узлу
- `gg` - Перейти к первому ходу
- `GG` - Перейти к последнему ходу

### Настройте это

Откройте настройки расширения, нажмите на поле горячей клавиши, нажмите любую клавишу, которую хотите.
Любая клавиша, любая комбинация. `n`/`p`? `,`/`.`? Решать вам.

**Режим потока**: Быстрые нажатия выстраиваются в очередь плавно.
**Режим прыжка**: Мгновенный отклик, максимальная скорость.
