# 시간 여행

긴 대화는 어수선해지기 마련입니다. 위아래로 스크롤하다 보면 현재 위치를 잃어버리곤 하죠.
Voyager는 당신의 대화를 타임라인으로 바꿔줍니다.

## 대화의 형태를 확인하세요

화면 오른쪽을 보세요.
각 노드는 메시지를 나타냅니다. 타임라인은 당신의 대화 리듬을 시각화합니다.

## 내비게이션의 해결책

- **순간 이동(Teleport)**: 노드를 클릭하여 해당 메시지로 즉시 이동합니다.
- **미리 보기(Peek)**: 마우스를 올려 이동하지 않고도 내용을 확인합니다.
- **북마크(Bookmark)**: 노드를 길게 눌러 **별표**를 표시하세요. 당신의 뇌를 위한 북마크와 같습니다.
- **레벨 (실험적)**: 노드를 우클릭하여 다양한 레벨(1-3)을 설정하거나 하위 노드를 접을 수 있습니다. 분기된 대화를 명확하게 정리하는 데 완벽합니다.
- **너비 조절**: 타임라인 안쪽 가장자리를 드래그하여 너비를 조절할 수 있습니다.
- **키보드**: 생각의 속도로 탐색하세요. 기본값은 `j`/`k`, `gg`/`GG`로 처음/마지막 턴으로 이동.

![타임라인 탐색](/assets/teaser.png)

## 별표: 영감의 순간을 북마크하세요

긴 대화 속에 금이 묻혀 있습니다. 별표로 중요한 순간을 고정하세요.

**사용 방법**: 타임라인의 아무 노드나 길게 누르세요(약 0.5초). 노드가 빛나면 — 해당 메시지에 별표가 표시된 것입니다.

- 다시 길게 누르면 별표가 해제됩니다.
- 별표가 표시된 노드는 강조 표시되어 한눈에 찾을 수 있습니다.
- 타임라인 헤더의 별표 아이콘을 클릭하면 **별표 기록** 패널에서 표시한 모든 메시지를 한곳에서 확인할 수 있습니다.

대화에 깃발을 꽂는 것과 같습니다 — 영감이 떠오른 순간, 완벽한 답변, 다음에 이어서 할 지점.

::: tip 팁
[클라우드 동기화](/guide/cloud-sync)를 활성화하면 별표가 기기 간에 동기화됩니다.
:::

## 키보드로 더 빠르게

마우스를 사용하고 싶지 않으신가요? 키보드를 사용하세요.

**Gemini에서 Vim 모드를 켜는 것과 같습니다.**

### 기본 단축키

- `k` - 이전 노드로 이동
- `j` - 다음 노드로 이동
- `gg` - 첫 번째 턴으로 이동
- `GG` - 마지막 턴으로 이동

### 맞춤 설정

확장 프로그램 설정을 열고 단축키 상자를 클릭한 다음 원하는 키를 누르세요.
어떤 키나 조합도 가능합니다. `n`/`p`? `,`/`.`? 당신의 선택입니다.

**플로우(Flow) 모드**: 빠르게 누르면 부드럽게 대기열에 추가됩니다.
**점프(Jump) 모드**: 즉각적인 반응, 최대 속도.
