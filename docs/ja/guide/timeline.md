# 時間旅行

長い会話は迷宮です。上へ下へとスクロールし、方向を見失います。
Voyager は、会話を一筋の「線」に変えます。

## リズムを見る

画面の右端を見てください。
そこにある点は、一つひとつが発言です。それは、あなたとAIの対話の脈動そのものです。

## ナビゲーション、一足飛び

- **瞬時に移動**：クリックした場所へ、即座に飛びます。もたつきません。
- **のぞき見**：マウスを乗せるだけで、ジャンプせずに中身を確認できます。
- **ブックマーク**：ノードを長押しして **スター** を付けましょう。脳のしおりです。
- **階層化 (実験的)**：ノードを右クリックして、レベル（1-3）を設定したり、子ノードを折りたたんだりできます。深い議論の枝葉を一目で把握できます。
- **幅の調整**：タイムラインの内側の端をドラッグして幅を調整できます。
- **ショートカット**：キーボードで高速移動。デフォルトは `j`/`k`、`gg`/`GG` で最初/最後のターンへ。

## スター：ひらめきをブックマーク

長い会話の中に、宝石が埋もれています。大事な瞬間をスターで留めましょう。

**使い方**：タイムライン上の任意のノードを長押し（約0.5秒）します。ノードが光れば、そのメッセージにスターが付きました。

- もう一度長押しでスターを解除。
- スター付きノードはハイライト表示され、ひと目で見つけられます。
- タイムラインヘッダーのスターアイコンをクリックすると、**スター履歴**パネルで全てのスター付きメッセージをまとめて確認できます。

会話にフラグを立てるようなものです——ひらめきの瞬間、完璧な回答、次回ここから再開したいポイント。

::: tip ヒント
[クラウド同期](/guide/cloud-sync)を有効にすると、スターはデバイス間で同期されます。
:::

## キーボードが生む速度

マウスを使いたくない？ キーボードがあります。

**まるで Gemini で Vim モードを使っているかのように。**

### デフォルト・ショートカット

- `k` - 前のノードへ移動
- `j` - 次のノードへ移動
- `gg` - 最初のターンへ移動
- `GG` - 最後のターンへ移動

### カスタマイズ

拡張機能の設定を開き、ショートカットの入力欄をクリックして、使いたいキーを押すだけです。
任意のキー、任意の組み合わせで。`n`/`p`？ `,`/`.`？ お好みでどうぞ。

**流動モード**では、連打するとアニメーションがキューに入って滑らかに再生されます。
**跳躍モード**では、即座に反応し、最高速度で移動します。
